# 199. 二叉树的右视图

## 解题思路


* 二叉树的层序遍历
* 每遍历一层 记录当前层的最右边的节点值

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<Integer> rightSideView(TreeNode root) {

        // 层序遍历的最右侧节点

        if(root == null){
            return new ArrayList<>();// 直接返回空
        }

        Queue<TreeNode> queue = new LinkedList<>();// 队列
        queue.offer(root);

        List<Integer> result = new ArrayList<>();// 保存结果

        while(!queue.isEmpty()){
            int sz = queue.size();
            int res = 0;// 保存最右边的节点

            for(int i = 0; i < sz; i++){
                TreeNode cur = queue.poll();// 队头出队

                res = cur.val;// 最后一定是右边的节点

                if(cur.left != null){
                    queue.offer(cur.left);
                }

                if(cur.right != null){
                    queue.offer(cur.right);
                }
            }

            result.add(res);// 记录结果
        }

        return result;
    }
}
```
