# 416. 分割等和子集


## 解题思路

* 01背包问题

```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for(int num:nums){
            sum += num;
        }

        if(sum % 2 != 0){
            return false;
        }

        sum = sum /2;
        int n = nums.length;

        boolean[][] dp = new boolean[n + 1][sum + 1];

        // base case 当背包的重量为0 直接都是true
        for(int i = 0; i <= n; i++){
            dp[i][0] = true;
        }

        // base case  没有物品 都是false
        for(int j = 0; j <= sum; j++){
            dp[0][j] = false;
        }

        for(int i = 1; i <= n; i++){
            for(int j = 1; j <= sum; j++){
                if(j - nums[i - 1] < 0){
                    // 说明已经装不满
                    dp[i][j] = dp[i - 1][j];
                }else{
                    // 装或者不装
                    dp[i][j] = dp[i - 1][j] || dp[i - 1][j - nums[i - 1]];
                }
            }
        }

        return dp[n][sum];
    }


}

```
