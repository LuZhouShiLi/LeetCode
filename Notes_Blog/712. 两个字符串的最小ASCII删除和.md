# 712. 两个字符串的最小ASCII删除和



## 解题思路

* 计算两个字符串的最小Ascii 删除和等价于计算两个字符串的最大公共子序列
* 然后使用总的Ascii值减去最大公共子序列的Ascii值


```java
class Solution {

    int[][] memo;// 记录子问题的解

    public int minimumDeleteSum(String s1, String s2) {
        // 需要被删除的字符 =  原字符串 -  最长公共子序列

        // dp[i][j] 表示子串 s1[0..i - 1] 和 s2[0..j - 1] 最小 ASCII 删除和

        int m = s1.length();
        int n = s2.length();
        int sum = 0;// 总的acsii值

        memo = new int[m + 1][n + 1];

        for(int i = 0; i < m; i++){
            sum += s1.charAt(i);
        }

        for(int i = 0; i < n; i++){
            sum += s2.charAt(i);
        }

        return sum - dp(s1,s2,m,n) * 2;

    }

    int dp(String s1,String s2,int m,int n){
        if(m < 0|| n < 0){
            return 0;
        }

        for(int i = 1; i <= m; i++){
            for(int j = 1; j <= n; j++){
                if(s1.charAt(i  -1) == s2.charAt(j - 1)){
                    // 说明找到一个公共字符  加上该字符的ascii值
                    memo[i][j] = memo[i - 1][j - 1] + ((int)s1.charAt(i - 1));
                }else{
                    memo[i][j] = Math.max(memo[i - 1][j],memo[i][j - 1]);
                }
            }
        }

        return memo[m][n];
    }
}
```
