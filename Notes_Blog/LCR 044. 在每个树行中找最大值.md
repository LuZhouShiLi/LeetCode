# LCR 044. 在每个树行中找最大值

## 解题思路

* 二叉树的层序遍历
* 遍历每一层的时候，记录最大值


```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<Integer> largestValues(TreeNode root) {
        // 二叉树的层序遍历 记录每一层的最大值

        if(root == null){
            return new ArrayList<>();
        }

        Queue<TreeNode> queue = new LinkedList<>();// 记录节点
        queue.offer(root);// 保存根节点

        List<Integer> result = new ArrayList<>();// 记录结果

        while(!queue.isEmpty()){
            int sz = queue.size();// 计算队列的大小
            int max = Integer.MIN_VALUE;// 保存最大值
            for(int i = 0; i < sz; i++){
                TreeNode cur = queue.poll();// 队头元素出队

                if(cur.val > max){
                    max = cur.val;
                }

                // 将当前节点的子节点进行入队
                if(cur.left != null){
                    queue.offer(cur.left);
                }

                if(cur.right != null){
                    queue.offer(cur.right);
                }
            }

            result.add(max);// 记录结果
        }

        return result;
    }
}
```
